# Kommentare

Sie können auf dieser Webseite Kommentare zu einzelnen Abschnitten hinterlassen. Diese Kommentare können nur von Ihnen und Ihrer Lehrperson gelesen werden.

Kommentare können zum Beispiel nützlich sein um sich zusätzliche Notizen zu machen oder Fragen für die nächste Stunde zu notieren.

Wenn Sie Ihre Maus über einen Abschnitt bewegen, erscheint rechts ein kleines Sprechblasen-Symbol. Klicken Sie darauf, um einen Kommentar zu diesem Abschnitt zu hinterlassen. Wenn Sie zusätzlich noch auf die drei Punkte unter dem Sprechblasen-Symbol klicken, können Sie die Farbe ändern.
